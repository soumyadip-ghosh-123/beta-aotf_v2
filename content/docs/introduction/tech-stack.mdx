## Framework & Runtime

| Technology | Version | Purpose |
|-----------|---------|---------|
| [Next.js](https://nextjs.org) | 16.x | Full-stack React framework (App Router) |
| [React](https://react.dev) | 19.x | UI rendering |
| [TypeScript](https://typescriptlang.org) | 5.9 | Type-safe development |
| [Turbopack](https://turbo.build/pack) | Built-in | Dev server bundler (`next dev --turbopack`) |

## Authentication & Security

| Technology | Purpose |
|-----------|---------|
| [Clerk](https://clerk.com) | Authentication provider (username/password, webhooks, session management) |
| Custom RBAC | 3-tier role system via `publicMetadata` |
| CSRF Protection | Origin/Referer validation on mutations |
| Rate Limiting | In-memory sliding-window limiter |
| NoSQL Injection Prevention | `mongoose.set('sanitizeFilter', true)` |
