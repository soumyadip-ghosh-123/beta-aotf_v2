## Base Path

```
/api/v1/admin
```

All endpoints in this section require **admin authentication**.

---

## Admin Management

### Create Admin

```
POST /api/v1/admin/admins
```

Creates a new admin account. **Super Admin only.**

**Request Body:**

```json
{
  "username": "johndoe",
  "email": "john@example.com",
  "firstName": "John",
  "lastName": "Doe",
  "role": "moderator"
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `username` | `string` | ✅ | Unique admin username |
| `email` | `string` | ✅ | Admin email address |
| `firstName` | `string` | ✅ | First name |
| `lastName` | `string` | ✅ | Last name |
| `role` | `"admin" \| "moderator"` | ✅ | Admin role level |

**Response:** `201 Created`

```json
{
  "success": true,
  "admin": {
    "_id": "...",
    "username": "johndoe",
    "email": "john@example.com",
    "name": "John Doe",
    "role": "moderator",
    "isActive": true,
    "createdAt": "2026-03-08T..."
  }
}
```

> A temporary password is generated and emailed to the new admin.

---

### List All Admins

```
GET /api/v1/admin/admins
```

Returns all admin accounts. Available to **Super Admin** and **Sub-Superadmin**.

**Response:** `200 OK`

```json
{
  "success": true,
  "admins": [
    {
      "_id": "...",
      "username": "superadmin",
      "role": "super_admin",
      "isActive": true,
      "isLocked": false,
      "createdAt": "2026-01-01T..."
    }
  ]
}
```

---

### Get Admin Details

```
GET /api/v1/admin/admins/[id]
```

Returns detailed information about a specific admin.

---

### Update Admin Permissions

```
PATCH /api/v1/admin/admins/[id]
```

Updates an admin's permissions or role. Sub-Superadmins can only modify **support** admins.

**Request Body:**

```json
{
  "permissions": {
    "canManagePosts": true,
    "canManageJobs": false,
    "canHandleEnquiries": true
  }
}
```

## Account Operations

### Deactivate Admin

```
POST /api/v1/admin/admins/[id]/deactivate
```

Deactivates an admin account. The admin can no longer log in.

---

### Reactivate Admin

```
POST /api/v1/admin/admins/[id]/reactivate
```

Reactivates a previously deactivated admin.

---

### Terminate Admin

```
POST /api/v1/admin/admins/[id]/terminate
```

Permanently terminates an admin account. **Super Admin only.** Records the termination in the admin document with `terminatedBy` and `terminatedAt`.

---

### Reset Password

```
POST /api/v1/admin/admins/[id]/reset-password
```

Resets an admin's password to a new temporary value. **Super Admin only.** The admin will be required to change their password on next login.

---

### Unlock Account

```
POST /api/v1/admin/admins/[id]/unlock
```

Unlocks an account that was locked due to failed login attempts. **Super Admin only.**

---

## Audit Logs

### Get Audit Logs

```
GET /api/v1/admin/audit-logs
```

Retrieves the admin action audit trail. Available to **Super Admin** and **Sub-Superadmin**.

**Query Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `page` | `number` | Page number (default: 1) |
| `limit` | `number` | Results per page (default: 20) |
| `action` | `string` | Filter by action type |
| `adminId` | `string` | Filter by acting admin |

**Response:** `200 OK`

```json
{
  "success": true,
  "logs": [
    {
      "adminUsername": "superadmin",
      "action": "admin.created",
      "targetType": "Admin",
      "targetIdentifier": "johndoe",
      "details": { ... },
      "ipAddress": "192.168.1.1",
      "createdAt": "2026-03-08T..."
    }
  ]
}
```
