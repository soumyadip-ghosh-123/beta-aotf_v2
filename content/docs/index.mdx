## Welcome to AOTF Docs

**Academy of Tutorials & Freelancers (AOTF)** is a full-stack platform that connects **Providers** (Tutors & Freelancers) with **Consumers** (Students, Parents & Clients). Built with Next.js, MongoDB, and Clerk, the platform offers end-to-end gig management — from posting requirements to application review, selection, and payment.

This documentation covers everything you need to understand, develop, and deploy the AOTF platform.

## Tech Stack at a Glance

| Layer | Technology |
|-------|-----------|
| Framework | Next.js 16 (App Router, Turbopack) |
| UI Library | HeroUI v2 + Tailwind CSS v4 |
| Database | MongoDB Atlas + Mongoose |
| Authentication | Clerk (Username/Password, Webhooks) |
| Payments | Razorpay (UPI, Card, Wallet) |
| Media | Cloudinary |
| Email | Resend |
| Monitoring | Sentry |
| Documentation | Fumadocs |
