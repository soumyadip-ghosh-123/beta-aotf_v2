# Jobs API

Job posts represent consumer requirements for freelancers and project contractors. They are created by admins and publicly listed for teacher-candidates to browse and apply.

## Job Statuses

| Status | Meaning |
|---|---|
| `open` | Accepting applications |
| `closed` | Assignment complete |
| `hold` | Temporarily paused |
| `cancelled` | Job cancelled |

## Work Types

| Value | Meaning |
|---|---|
| `job` | Permanent or part-time employment |
| `project` | Time-limited project work |

## List Jobs

```http
GET /api/v1/jobs
```

Public endpoint. No authentication required.

**Query Parameters**:

| Param | Type | Description |
|---|---|---|
| `page` | `number` | Page (default: 1) |
| `limit` | `number` | Per page (default: 20) |
| `status` | `string` | Filter by status |
| `workType` | `string` | Filter: `job`, `project` |
| `locationType` | `string` | Filter: `remote`, `onsite`, `hybrid` |
| `gender` | `string` | Filter: `male`, `female`, `both`, `all` |

**Response 200**:
```json
{
  "jobs": [
    {
      "jobId": "JOB-001",
      "workType": "project",
      "title": "Frontend Developer for E-commerce Site",
      "clientName": "Raj Enterprises",
      "source": "Referral",
      "companyType": "company",
      "locationType": "remote",
      "location": "Remote",
      "timing": "Flexible",
      "gender": "all",
      "status": "open",
      "commissionBasis": "project_value",
      "academyCommissionPercentage": 10,
      "createdAt": "2026-01-01T00:00:00.000Z"
    }
  ],
  "total": 18,
  "page": 1
}
```

Note: `clientName` and `phoneNumber` are **not included** in public list responses. They are only shown to authenticated providers after they apply and are shortlisted.

## Get Single Job

```http
GET /api/v1/jobs/:jobId
```

Public endpoint. `jobId` is the human-readable ID (e.g., `JOB-001`), not the MongoDB `_id`.

## Create Job

```http
POST /api/v1/jobs
Authorization: Bearer <admin_clerk_jwt>
Content-Type: application/json
Origin: https://aotf.in
```

**Permission required**: `canCreateJobPosts`

**Request Body**:
```json
{
  "workType": "project",
  "title": "Frontend Developer for E-commerce Site",
  "clientName": "Raj Enterprises",
  "phoneNumber": "9876543210",
  "enquiryId": "6843f2a1b4e5c9d0e8f12345",
  "source": "Referral",
  "companyType": "company",
  "locationType": "remote",
  "location": "Remote",
  "timing": "Flexible",
  "experience": "2+ years",
  "gender": "all",
  "salary": null,
  "requiredQualification": "B.Tech / BCA",
  "projectType": "one-time",
  "budget": "₹15,000–₹20,000",
  "duration": "1 month",
  "brief": "Build a responsive React storefront",
  "commissionBasis": "project_value",
  "academyCommissionPercentage": 10
}
```

**Required fields**: `workType`, `title`, `clientName`, `phoneNumber`, `source`, `companyType`, `locationType`, `location`, `timing`, `gender`, `commissionBasis`, `academyCommissionPercentage`

**Response 201**: Created job document.

## Update Job

```http
PATCH /api/v1/jobs/:jobId
Authorization: Bearer <admin_clerk_jwt>
Content-Type: application/json
Origin: https://aotf.in
```

**Permission required**: `canEditPosts`

All fields optional.

**Response 200**: Updated job document.

## Delete Job

```http
DELETE /api/v1/jobs/:jobId
Authorization: Bearer <admin_clerk_jwt>
Origin: https://aotf.in
```

**Permission required**: `canDeletePosts`

**Hard delete**.

## Commission Fields

AOTF earns a commission on placed jobs:

| Field | Type | Description |
|---|---|---|
| `commissionBasis` | `"first_month"` \| `"project_value"` | What the commission % applies to |
| `academyCommissionPercentage` | `number` | AOTF's cut (e.g., `10` = 10%) |

This data is used for invoice generation and payment tracking.

| `gender` | `male`, `female`, `both`, `all` |
