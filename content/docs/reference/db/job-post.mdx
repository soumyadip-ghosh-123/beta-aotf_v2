## PostLedger

**Collection**: `post_ledgers`  
**Source**: [`lib/models/PostLedger.ts`](file:///d:/aotf_v2/lib/models/PostLedger.ts)

The assignment tracking record for each tuition post. This is the **source of truth** for the "Tuitions" Google Sheet tab. It contains the full picture of a tuition lifecycle: from open post → teacher assigned → payment status → invoice.

### Key Fields (beyond Post snapshot)

| Field | Type | Description |
|---|---|---|
| `serialNumber` | `number` | Sheet row serial number |
| `postStatus` | `enum` | `open`, `assigned`, `closed` |
| `assignedTeacherId` | `string` | Clerk ID of assigned teacher |
| `assignedTeacherName` | `string` | Snapshot name at assignment time |
| `assignedTeacherPhone` | `string` | Snapshot phone at assignment time |
| `assignedTeacherUsername` | `string` | For profile link |
| `teacherGender` | `string` | Resolved from Profile at assignment |
| `assignedAt` | `Date` | When teacher was assigned |
| `teacherDemoDate` | `Date` | DC date from Application |
| `startingDate` | `Date` | Admin-entered tuition start date |
| `cancelledOrNot` | `boolean` | Whether Post was cancelled |
| `requirement` | `string` | Human-readable summary of student needs |
| `teacherHasBeenPaid` | `boolean` | Whether AOTF paid the teacher |
| `teacherPaymentDate` | `Date` | When AOTF paid the teacher |
| `invoiceGenerated` | `boolean` | Whether invoice was created |
| `invoiceId` | `string` | Invoice ID |
| `paymentStatus` | `enum` | `unpaid`, `paid`, `partial` (guardian payment) |
| `paymentDate` | `Date` | When guardian paid |
| `paymentAmount` | `number` | Amount paid by guardian |
| `sheetRowIndex` | `number` | Row in "Tuitions" sheet |
| `statusHistory` | `array` | Status change records with timestamps |
| `teacherChangeCount` | `number` | How many times teacher was reassigned |

### Google Sheets Sync

Every `PostLedger` upsert writes to the **"Tuitions"** tab:

```ts
// lib/services/postLedger.service.ts  
await sheets.spreadsheets.values.update({
  spreadsheetId: process.env.GOOGLE_SHEETS_SPREADSHEET_ID,
  range: `Tuitions!A${sheetRowIndex}:AZ${sheetRowIndex}`,
  valueInputOption: "RAW",
  requestBody: { values: [mapLedgerToRow(ledger)] },
});
```
