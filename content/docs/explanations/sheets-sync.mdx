# Google Sheets Sync

AOTF writes enriched snapshots of posts, enquiries, and admin data to **Google Sheets** so the team can view pipeline data in a familiar spreadsheet without granting full admin access.

## How It Works

1. MongoDB remains the **source of truth** (`PostLedger`, `EnquiryLedger`, etc.)
2. Mongoose hooks and services update ledger collections on create/update
3. `lib/googleSheets.ts` pushes rows to configured tabs via a service account
